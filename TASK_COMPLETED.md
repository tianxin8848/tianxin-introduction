## 核心功能说明

### 图片轮播实现
- 使用 React Hooks (useState, useEffect) 管理状态
- 4张图片预加载，确保流畅切换
- CSS 透明度过渡实现淡入淡出效果
- 支持自动轮播和手动切换

### 多语言切换
- 语言偏好自动保存到浏览器本地存储
- 所有文本内容通过 JSON 文件管理
- 支持动态语言切换，无需刷新页面

### 动态路由管理
- 路由配置集中管理 (`src/routes/index.js`)
- 易于扩展和维护
- 导航栏自动根据路由配置生成

### 鼠标活动检测
- 使用 useRef 管理定时器
- 监听鼠标移动和点击事件
- 自动切换导航栏和时间显示
